Tasker app

Task Manager es una aplicación web que funciona en cualquier dispositivo y navegador, sin necesidad de descargar ni instalar nada. Solo tienes que registrarte con tu correo electrónico y contraseña, y empezar a crear tus listas de tareas. Además, Task Manager es una aplicación gratuita, sin anuncios ni limitaciones.
